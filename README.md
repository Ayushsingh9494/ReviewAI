📦 AI ReviewAI: The Ultimate Review Aggregator
An elegant, AI-driven platform that helps users find the best products by aggregating and summarizing realistic consumer reviews. Built for the modern web with a focus on speed, aesthetics, and intelligence.

✨ Key Features
🤖 Smart AI Analysis: Powered by the Vercel AI SDK and Gemini, providing deep insights into product strengths, weaknesses, and comparisons.
💎 Premium UI/UX: A glassmorphic design featuring interactive mouse-tracking "Spotlight" effects and 3D holographic Prism loaders.
📂 Session Management: Full chat history and persistent sessions powered by Supabase, allowing users to never lose their research.
🔒 Secure Auth: A complete authentication flow featuring a modern, interactive Sign-In/Sign-Up experience.
📊 Dynamic Comparisons: Automatically generates beautifully formatted markdown tables for side-by-side gadget comparisons.
🌙 Seamless Dark Mode: A unified design system that respects user preferences without any "white flash" on load.

🛠️ Tech Stack
Frontend: Next.js (App Router), Vanilla CSS (Custom Design System)
AI: Vercel AI SDK + Google Gemini API
Database & Auth: Supabase
Deployment: Netlify Ready
